# Greenden-Tailwind

🌱 GREENDEN – Responsive Front-End Website
GREENDEN is a modern, responsive front-end website built using HTML and CSS. It showcases a clean and elegant design for a plant and flower business, offering both artificial and natural plants for interior decoration. The project demonstrates strong fundamentals in layout structuring, responsive design, and user interface styling.

✨ Features
Fully responsive design that adapts to all screen sizes (mobile, tablet, desktop)

Sticky navigation bar with hover effects for smooth user interaction

Attractive hero section with promotional text and call-to-action buttons

About Us section highlighting business features (Large Assortment, Fast Shipping, 24/7 Support)

Organized product showcase grid with hover animations

Clean typography and consistent color palette for a professional look

🧠 Purpose
This project was created to practice and demonstrate front-end development skills using HTML5 and CSS3, focusing on responsive layouts and modern UI design principles. It serves as a portfolio piece for web development learners and professionals.

⚙️ Technologies Used
HTML5

CSS3

Tailwind CSS (for styling and responsiveness)

📱 Responsive Design
Mobile-friendly layout with adaptive navigation

Grid-based product section that adjusts seamlessly across devices
